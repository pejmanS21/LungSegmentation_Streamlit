# Records

Each **directory** contains a `mlflow` folder that you can access to model results by run this command in your `Terminal`:

    mlflow ui

**also** you can run `.sh` file in mentioned directory to get the same result.

    chmod +x mlflowui.sh
    ./mlflowui.sh
